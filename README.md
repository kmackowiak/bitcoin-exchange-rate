# Bitcoin exchange rate
Application that is a conversion rate of bitcoin to PLN.

### Tech used
* HTML
* CSS
* Pure JavaScript
* AJAX

### API sources 
* Bitcoin USD rate:
https://api.coinpaprika.com/
* USD to PLN rate: 
http://api.nbp.pl/
